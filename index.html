Stylestorm Project Overview

This project will allow people to get instant feedback on whether a style is appropriate for various situations. Stylestorm Project is being created as a stand in for those who don't have or are lacking a social consciousness and need an opinon about what to wear for a specific event or occasion. Example Design website Example CSS Zen Garden from HTML Me Something

Features

  • User login: Users will be able to log in, either by creating accounts in the system, or by using a third-party authentication provider (e.g. Google Authentication). All access to the site will be restricted to logged-in users. Launchcode User Authentication
  • Create and edit posts: Users will be able to create new posts in the system by entering in a mixture of required (title, description, etc) and optional fields (photos, videos etc).
  • View random posts: Users will be able to view a listing of recent and or popular posts, and by clicking on a post, view details about that post.
  • Volunteer tracking: Users will be able to add users to follow
  • Delete posts: Users will be able to delete posts.
